## 3. 开发经验

- 按模块全栈开发能尽早发现接口字段、权限和页面体验问题。
- 库存流水是库存追溯的核心，不能只更新库存余额。
- 离线上传应保存对象存储地址，不依赖本地临时目录。

## 4. 面试点

### Q1：为什么 WMS 要前后端一起开发？

答：WMS 包含大量表格、表单、上传和权限场景，前后端一起开发可以及时验证接口设计和数据权限。

### Q2：如何避免离线上传重复导入？

答：对 `file_hash` 或 `upload_batch_no` 建唯一约束，重复上传时返回已有任务或拒绝重复处理。
