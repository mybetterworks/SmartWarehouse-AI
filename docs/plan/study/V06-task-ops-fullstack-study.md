## 4. 面试点

### Q1：多实例定时任务如何避免重复执行？

答：可以使用分布式锁、XXL-Job、K8s CronJob 或 Leader Election，保证同一任务同一时刻只有一个实例执行。

### Q2：为什么排行榜用 Redis ZSet？

答：ZSet 同时维护成员和分数，支持快速更新、排序和范围查询，适合实时排行。
